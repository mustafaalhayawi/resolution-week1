# resolution-week1
My submission for week 1 of the hardware track of https://resolution.hackclub.com.

## Description
This circuit consists of a DC voltage source, resistors, capacitors and LEDs and creates a fading LED pattern where each LED fades longer than the one before.

## How it works
The circuit consists of a 7.5V DC voltage source, a switch, and 3 capacitors of varying capicitance connected together in series. Each capacitor has connected to it in parallel an LED and resistor. When the switch is opened, the capacitors discharge, lighting up the LEDs. Since the capacitors have varying capacitors, hence storing different amounts of charge and they light up the LED for different amounts of time.
