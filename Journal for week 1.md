Originally I planned on creating the light effect so that it turns on and off automatically but I realised this wouldn't be possible without active components.

Then I settled on creating a light effect that had to be turned on/off manually using a switch. At first I faced some issues because I connected multiple capacitors and LEDs in parallel but I fixed this by connecting the capacitors in series and each LED to a capacitor in parallel.

I learnt about RC time constants and getting the light effect to work was satisfying.

Total time spent: ~1.5 hours
